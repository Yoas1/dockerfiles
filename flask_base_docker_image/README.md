# flask_base_docker_image

## For edit web-app files:
</br>
Clone this repo, and create volume:</br>

```
docker container run -it -v <path/to/flask-app:/App -p 5000:5000 yoas1/flask-base:1.0
```


Go to: http://localhost:5000
