# Apache-Airflow<br><br>

Run with command:<br>
```
docker container run -it --name airflow -p 8080:8080 yoas1/apache-airflow:v2.0.1
```
Add volumes to logs and dags:</br>
**Logs:** /root/airflow/logs
**Dags:** /root/airflow/dags
</br></br>

Go to: http://localhost:8080</br>
**User:** admin</br>
**Password:** admin
